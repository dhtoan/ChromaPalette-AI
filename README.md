# ChromaPalette AI (Web Edition) 🎨

A lightweight, professional-grade color extraction tool that runs entirely in your browser.

## ✨ Features
- **Zero Server-Side**: Uses client-side K-Means clustering for maximum privacy and speed.
- **Smart Extraction**: Implements K-Means algorithm to find mathematically optimal color centroids.
- **Responsive UI**: Beautifully designed with Vue 3 and Tailwind CSS.
- **Developer Friendly**: Instant copy-to-clipboard for HEX codes.

## 🛠 Tech Stack
- **Framework**: [Vue.js 3](https://vuejs.org/) (Composition API)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Algorithm**: [ml-kmeans](https://github.com/mljs/kmeans) (K-Means Clustering)
- **Deployment**: GitHub Pages

## 🚀 Deployment Instructions (GitHub Pages)

1. Create a new repository on GitHub.
2. Upload `index.html` to the repository.
3. Go to **Settings** > **Pages**.
4. Under **Build and deployment** > **Branch**, select `main` and `/ (root)`.
5. Click **Save** and wait a minute for your live site!
